# Servicio gratuito

En este artículo, te explicaremos cómo crear y mantener un servidor gratuito en Whost.sh para Minecraft y bots de Discord. Sigue los pasos a continuación para obtener tu propio servidor en unos pocos minutos.

### Creación del Servidor Gratuito

1. Accede a la página del [servicio gratuito](https://panel.whost.sh/freeservers).

2. En esta página, se mostrarán los diferentes tipos de servidores gratuitos que ofrecemos. Examina las opciones disponibles y selecciona el tipo de servidor que mejor se ajuste a tus necesidades y preferencias.

3. Una vez que hayas elegido el tipo de servidor, comprueba si hay suficiente espacio disponible en el nodo para crear tu servidor. Si hay espacio suficiente, procede a crearlo haciendo clic en el botón "Crear mi servidor".

4. Una vez que se haya creado tu servidor, serás redirigido automáticamente a la consola del mismo. En esta página, podrás ver el estado de tu servidor, así como el uso de los recursos.

### Mantener el Servidor Gratuito

Una vez que hayas creado tu servidor gratuito, es importante mantenerlo activo para seguir disfrutando de los servicios. Para ello, sigue estos pasos:

1. Después de crear el servidor, verás un contador que indica el tiempo restante para que el servidor siga activo. (Ver imagen adjunta).

![Contador de servidor gratuito](https://cdn.whost.sh/docs/images/free_server_counter.png)

2. El contador se inicia con 24 horas y disminuirá con el tiempo. Si deseas mantener el servidor activo, es necesario renovarlo antes de que el contador llegue a 0.

3. Para renovar el servidor, simplemente haz clic en el botón de "Renovar" que estará disponible cuando el contador esté en marcha.

4. Al renovar el servidor, se agregarán 24 horas adicionales al contador, y el servidor seguirá funcionando sin problemas.

5. Si por alguna razón no se renueva el servidor antes de que el contador llegue a 0, el servidor se suspenderá automáticamente.

6. En caso de que el servidor se haya suspendido, tendrás 48 horas para renovarlo antes de que se elimine permanentemente.

## ¡Importante! 

Es fundamental que los usuarios creen sus propios backups de los servicios gratuitos, ya que WHOST no crea backups para los servidores gratuitos.

Recuerda que al seguir estos pasos, podrás disfrutar de tu servidor gratuito de Minecraft o bot de Discord de manera continua y sin interrupciones.
