# Servicio de pago

Si deseas obtener un servidor de mayor rendimiento y funcionalidades adicionales, puedes adquirir uno de nuestros servidores de pago en https://billing.whost.sh/. Aquí te explicamos cómo comprar y mantener tu servidor de pago:

### Comprar un Servidor de Pago

1. Accede a la [tienda](https://billing.whost.sh/) en tu navegador web.

2. Inicia sesión con tu cuenta o crea una nueva si aún no tienes una.

3. Una vez dentro del panel de control de facturación, navega hacia la sección de "Comprar Servidor" o "Nuevos Servidores".

4. Aquí verás una lista de los diferentes tipos de servidores disponibles para la compra. Selecciona el que mejor se adapte a tus necesidades y preferencias.

5. Personaliza las opciones del servidor, como la cantidad de RAM, la capacidad de almacenamiento y otros recursos, según tus requerimientos.

6. Añade el servidor al carrito y procede al proceso de pago.

7. Completa el proceso de pago proporcionando la información necesaria y seleccionando el método de pago preferido.

8. Una vez que el pago se haya completado, recibirás una confirmación y tu servidor estará listo para su uso.

### Mantener el Servidor de Pago

A diferencia de los servidores gratuitos, los servidores de pago no se renuevan automáticamente. Debes mantener el pago de manera mensual para seguir utilizando los servicios. Aquí te explicamos cómo hacerlo:

1. Al adquirir un servidor de pago, recibirás una factura correspondiente al primer mes de servicio.

2. Asegúrate de pagar la factura antes de que expire el plazo indicado en la misma, generalmente mensual.

3. Si la factura no se paga a tiempo, el servidor se suspenderá automáticamente hasta que se realice el pago.

4. Una vez que el pago se haya efectuado, el servidor se reactivará y podrás seguir utilizándolo sin interrupciones.

5. Es importante recordar pagar puntualmente cada mes para mantener el servidor en funcionamiento.

6. Si deseas cancelar el servicio, asegúrate de hacerlo antes de la fecha de vencimiento de la próxima factura para evitar cargos adicionales.
